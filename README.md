# 天璇智能云 - 前端项目

## 项目简介

这是天璇智能云平台的前端界面项目，支持本地浏览器直接打开，无需构建工具。

## 快速开始

1. **打开方式**：双击 `index.html` 文件，或用浏览器打开
2. **无需服务器**：所有资源通过CDN加载，数据在本地模拟

## 使用说明

### 查看模型详情

点击表格操作列的「详情」按钮，查看模型完整信息。

### 手动扩缩容

1. 点击「更多」→「扩缩容」
2. 在弹窗中调整实例数（支持加减按钮）
3. 点击「确定」保存

### 配置自动扩缩容

1. 点击「更多」→「扩缩容」
2. 展开「自动扩缩容配置」面板
3. 勾选「启用自动扩缩容」
4. 设置参数：
   - 最小/最大实例数
   - 扩容阈值（CPU > %）
   - 缩容阈值（CPU < %）
   - 冷却时间（秒）
5. 点击「确定」保存

启用后，表格「实例数」列会显示🔄图标标识。
